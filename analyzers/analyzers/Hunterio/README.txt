### Hunter.io 
[Hunter.io](https://hunter.io/) Domain Search requires a domain name and returns a set of data about the organisation, the email address found and additional information about the people owning those email addresses.


#### Requirements
You need a valid Hunter.io Community API integration subscription to use the analyzer.
Free plan account are limted to 10 email addresses in results.

- Provide your API key as values for the `key` parameter.
